Когда делается GET запрос через браузер, запрос блокируется по политике CORS,
 так как нет разрешающего заголовка Access-Control-Allow-Origin для origin

При изпользовании Node запрос обычно проходит,
 потому что в Node нет браузерной CORS-политики (CORS — ограничение именно браузера).

Для API geoiplookup.io, результаты аналогичны
